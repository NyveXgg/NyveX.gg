# Plot Plugin – Spielerbefehle (`/p` / `/plot`)

Diese Datei listet **ausschließlich Spielerbefehle** für das Plot Plugin auf. Keine Admin-/Staff-Befehle, keine Berechtigungsinformationen — nur die Befehle, die Spieler verwenden.

---

## Kurze Hinweise
- `/p` und `/plot` sind Alias-Befehle und funktionieren meist identisch.
- Viele Befehle wirken auf das Plot, in dem du dich gerade befindest, oder auf dein eigenes Plot.
- Platzhalter wie `<Spieler>` ersetzen durch den Spielernamen.

---

## Spielerbefehle (`/p`)

- `/p home`  
  Teleportiert dich zu deinem Plot-Spawn / Plot-Home.

- `/p claim`  
  Beansprucht ein freies Plot an deiner aktuellen Position.

- `/p auto`  
  Beansprucht automatisch ein freies Plot in der Nähe (schnelle Claim-Option).

- `/p abandon`  
  Gibt dein aktuelles Plot frei (Plot wird wieder verfügbar).

- `/p add <Spieler>`  
  Fügt `<Spieler>` als Mitbauer/Member zu deinem Plot hinzu.

- `/p remove <Spieler>`  
  Entfernt `<Spieler>` aus deinem Plot (entzieht Baurechte).

- `/p members`  
  Zeigt die aktuellen Mitglieder (Mitbauer) deines Plots an.

- `/p info`  
  Zeigt Informationen über das Plot an (Größe, Besitzer, Mitglieder, Status).

- `/p toggle`  
  Schaltet bestimmte Plot-Optionen (z. B. Build-Modus, PVP, usw.) um — abhängig von Server-Einstellungen.

- `/p biome <Name>`  
  Ändert das Biome deines Plots (falls vom Server erlaubt).

- `/p visit <Spieler>`  
  Teleportiert dich zum Plot von `<Spieler>` (sofern dieser ein Plot besitzt und Besuch erlaubt).

---

## Spielerbefehle (`/plot`)

- `/plot home`  
  Teleportiert dich zu deinem Plot-Spawn / Plot-Home.

- `/plot claim`  
  Beansprucht ein freies Plot an deiner aktuellen Position.

- `/plot auto`  
  Beansprucht automatisch ein freies Plot in der Nähe.

- `/plot abandon`  
  Gibt dein Plot auf und macht es wieder frei.

- `/plot add <Spieler>`  
  Fügt `<Spieler>` zu deinem Plot hinzu.

- `/plot remove <Spieler>`  
  Entfernt `<Spieler>` von deinem Plot.

- `/plot members`  
  Zeigt die Mitglieder deines Plots an.

- `/plot info`  
  Zeigt Informationen über das Plot an.

- `/plot toggle`  
  Schaltet Plot-Optionen ein/aus (je nach Server-Konfiguration).

- `/plot biome <Name>`  
  Ändert das Biome deines Plots (wenn erlaubt).

- `/plot visit <Spieler>`  
  Teleportiert dich zum Plot von `<Spieler>`.

---

## Kurz-FAQ (Spieler)

- **Wie claim ich schnell ein Plot?** → Nutze `/p auto` oder `/plot auto`.  
- **Wie lade ich Leute zum Bauen ein?** → `/p add <Spieler>` (oder `/plot add <Spieler>`).  
- **Wie verlasse ich mein Plot?** → `/p abandon` (oder `/plot abandon`).  
- **Wie gehe ich zu einem anderen Plot?** → `/p visit <Spieler>` (oder `/plot visit <Spieler>`).
